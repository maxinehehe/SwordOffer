# 变相跳台阶

```
一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。
```

### 考点：循环和递归

## 思路：

```
1 个 台阶 有 1 种跳法 [1]
2 个 台阶 有 2 种跳法 [11,2]   - 1_(剩一个台阶 只有一种跳法) +1一跃而达 
3 个 台阶 有 4 种跳法 [111,12,21,3] 1_ _(剩两个台阶 2 种跳法)2_(一种)+1
4 个 台阶 有 8 种跳法 [1111,112,211,121,22,13,31,4] 。。。
...
则可以推出， 后面的结果 是 2^(n - 1) .

```



## 代码

```java
public class Solution {
    public int JumpFloorII(int target) {
        int res = 1;
        if(target == 1)
            return 1;
        for(int i = 1; i < target; i++){
            res *= 2;
        }
        return res;
    }
}
```
