# 两个链表的第一个公共结点

### 题目描述
```
输入两个链表，找出它们的第一个公共结点。（注意因为传入数据是链表，所以错误测试数据的提示是用其他方式显示的，保证传入数据是正确的）
```

### 解题思路
```
其中题意需要认真理解
公共节点 表示 当前节点相等 并且next指向的下一个节点也相等。

第一种情况：
    1->2->3->8->7->null
    4->3->8->7->null(或者是 8->7 / 1->2->3)
    判断两个 链表的长度 
    首先 如果 链表的长度 前面不相等， 那么就继续保持两个链表长度一致再找相等的
    如果相等就直接返回
    
    一种方法是遍历两个链表统计长度， 然后第二次遍历中 保持先从长的链表开始，到两链表长度一致再开始 一同遍历找相同
    另一种方法就是 两个链表都走 如果有一个先遍历完了就将其其指向另一个链表的头结点，剩下的一个遍历完也指向另一个头结点
    这样 相当于长的链表夺走了k 步（k即为两链表长度差）， 然后再进行遍历找公共节点。
```

### 代码

```java
/*
public class ListNode {
    int val;
    ListNode next = null;

    ListNode(int val) {
        this.val = val;
    }
}*/
public class Solution {
    // 思路 
    /**
    第一种情况：
    1->2->3->8->7->null
    4->3->8->7->null(或者是 8->7 / 1->2->3)
    判断两个 链表的长度 
    首先 如果 链表的长度 前面不相等， 那么就继续保持两个链表长度一致再找相等的
    如果相等就直接返回
    
    一种方法是遍历两个链表统计长度， 然后第二次遍历中 保持先从长的链表开始，到两链表长度一致再开始 一同遍历找相同
    另一种方法就是 两个链表都走 如果有一个先遍历完了就将其其指向另一个链表的头结点，剩下的一个遍历完也指向另一个头结点
    这样 相当于长的链表夺走了k 步（k即为两链表长度差）， 然后再进行遍历找公共节点。
    */
    public ListNode FindFirstCommonNode(ListNode pHead1, ListNode pHead2) {
         ListNode p1 = pHead1;
         ListNode p2 = pHead2;
        while(p1!=p2){  // 边走的过程还要判断是否一开始就能找到公共结点
            p1 = p1 == null ? pHead2:p1.next;
            p2 = p2 == null ? pHead1:p2.next;
        }
        return p1; // 此时返回 p1 或 p2都可以 公共结点
    }
}
```
