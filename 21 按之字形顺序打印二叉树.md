# 按之字形顺序打印二叉树

## 题目描述

请实现一个函数按照之字形打印二叉树，即第一行按照从左到右的顺序打印，第二层按照从右至左的顺序打印，第三行按照从左到右的顺序打印，其他行以此类推。

```解题思路
这是对二叉树层序遍历的一种应用：
给定一颗二叉树：
         8
 	   /  \
 	  6   10
 	 / \  / \
 	5  7 9 11
 层序遍历 为 ： 8 6 10 5 7 9 11
 那么按照之字形打印则是：8 10 6 5 7 9 11 
 规则就是根据层序遍历一行按照正序打印，下一行按照倒序打印，再下一行按照正序打印...如此循环下去，直到打印完毕。
 整体思路依然是层序遍历的变种。
 具体：使用队列配合存储容器的思想，使用一个队列进行判断每一层的节点数，即一次只放入队列中一层节点数，然后依次出队列根据出队列的元素 再搜集其左右子树节点，并判断此时是正序打印还是倒序打印，然后添加进存储容器。
 
```

## 代码实现

```java
import java.util.ArrayList;

/*
public class TreeNode {
    int val = 0;
    TreeNode left = null;
    TreeNode right = null;

}

import java.util.LinkedList;
public class Solution {
    public ArrayList<ArrayList<Integer> > Print(TreeNode pRoot) {
        LinkedList<TreeNode> queue = new LinkedList<>();
//        stack.poll();//  移除并出
//        Stack<TreeNode> stack = new Stack<>();
        ArrayList<ArrayList<Integer>> result = new ArrayList<>();

        queue.add(pRoot);  // 入一个栈元素
        boolean flag = true;   // 用于更改存入方式 左右
        while(!queue.isEmpty()){ // 当栈不空时
            int size = queue.size();   // 当前元素长度
            ArrayList<Integer> list = new ArrayList<>();
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();  // 出队列
//                queue.pop()
                // 必须要判断是否为空
                if(node == null)
                    continue;
                if(flag){ // 如果是 层遍历 是正序遍历就直接添加
                    list.add(node.val);
                }else {   // 否则就反着来
                    list.add(0, node.val);
                }
                // 这里需要将其左右子节点添加进去
                queue.offer(node.left);
                queue.offer(node.right);
            }
                // 同样 需要判断list不为0然后就直接添加
                if(list.size()!=0)
                    result.add(list);
                // 然后进行翻转 添加标记 flag
                flag = !flag;
        }
        return result;
    }

}
```
